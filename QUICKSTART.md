# Quick Start Guide

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Open your browser:**
   The game will automatically open at `http://localhost:3000`

## Game Flow

1. **Menu** - Start a new game or continue your adventure
2. **Introduction** - Learn about your quest as a Yoik Guardian Apprentice
3. **World Map** - Navigate to different challenges across S치pmi
4. **Challenges** - Complete 7 different educational challenges:
   - Language Puzzle (S치mi vocabulary)
   - Reindeer Herding (herd management)
   - Duodji Crafting (traditional patterns)
   - Environmental Challenge (ecosystem balance)
   - Yoik Puzzle (rhythm and music)
   - History Timeline (S치mi history)
   - Moral Choices (cultural values)
5. **Final Mission** - Restore the Runebomme with all collected artifacts

## Controls

- **Mouse**: Click to interact with buttons, drag items, select options
- **Arrow Keys**: Used in Reindeer Herding challenge to guide the herd
- **Click and Drag**: Used in Duodji Crafting and History Timeline challenges

## Features

- **Progress Tracking**: Your progress is automatically saved to localStorage
- **Cultural Knowledge Tokens**: Earn tokens by completing challenges
- **Artifact Collection**: Collect 7 artifacts to unlock the final mission
- **Educational Content**: Learn about S치mi culture, history, and traditions

## Building for Production

```bash
npm run build
```

The built files will be in the `dist/` directory, ready for deployment.
