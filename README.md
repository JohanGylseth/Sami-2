# Sámi Quest: Guardians of Sápmi

An educational game for Norwegian students (5th-7th grade) that teaches about Sámi culture, history, language, environment, and traditional knowledge.

## About the Game

You play as a Yoik Guardian Apprentice, tasked with collecting magical artifacts scattered across Sápmi. Each artifact represents important cultural knowledge, and to retrieve them, you must complete various challenges that teach you about:

- Sámi language and vocabulary
- Traditional reindeer herding
- Duodji (traditional handicrafts)
- Environmental stewardship
- Joik and musical traditions
- Sámi history and rights
- Cultural values and respect

## Technology

- **Framework**: Phaser.js 3.x
- **Build Tool**: Vite
- **Language**: JavaScript (ES6+)

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Game Structure

- **Menu Scene**: Main menu with start/continue options
- **Intro Scene**: Story introduction and quest setup
- **World Map Scene**: Navigate to different challenges
- **Challenge Scenes**: 7 different educational challenges
- **Final Mission Scene**: Restore the Runebomme

## Educational Content

All content is designed to be:
- Age-appropriate for 10-13 year olds
- Culturally sensitive and accurate
- Respectful of Sámi traditions
- Free from stereotypes

## Cultural Sensitivity

This game has been designed with careful attention to:
- Accurate representation of Sámi culture
- Respectful portrayal of joik (no imitation of sacred content)
- Contemporary Sámi representation
- Diverse character representation
- Age-appropriate historical context

## Assets

Currently uses programmatic graphics (shapes, colors, text). Placeholder assets are generated at runtime. Future versions may include custom graphics and audio.
